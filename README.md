# Ampbox - http://ampbox.io/

## Jam in your browser!

AmpBox is a browser based virtual pedalboard / ampsimbuilt using React, Tailwind & ToneJS. This is experimental as of now and might be unstable.

## Commands to run the project

```
npm i
npm start
```
